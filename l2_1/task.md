Что выведет программа?  

```Go
package main

import "fmt"

func main() {
  a := [5]int{76, 77, 78, 79, 80}
  var b []int = a[1:4]
  fmt.Println(b)
}
```

Программа создаст массив а с 5 элементами, используется короткая инициализация :=  
Далее создается срез b через var и ему присваивается срез из элементов массива а, с 1-го элемента включительно и по 4-й не включительно  
В Итоге програма выведет на экран срез b с числами {77, 78, 79}  
И в дополнение, массив a и срез b имеют общую память
