Что выведет программа?  

Объяснить вывод программы.  

```
// Происходит печать цифр с 0 до 9 и 
// fatal error: all goroutines are asleep - deadlock!  

func main() {
  ch := make(chan int)
  // создается небуферизированный канал ch и в него идет запись значений int в цикле for  
  
  go func() {
    for i := 0; i < 10; i++ {
      ch <- i
    }

    // Проблема в том что канал не закрывается после записи значений  
    // и остается открытым, нету close(ch)
    // Завершение горутины не означает закрытие канала  
    // ниже цикл for n := range ch { будет выводить значение пока канал не будет закрыт  
    // горутина отправляет число 9 в канал и завершает свою работу  
    // range ch продолжает ждать следующие числа,  
    // но Go видет что нету работающих горутин    
    // и происходит ошибка fatal error: all goroutines are asleep -  deadlock!  

    // Решение проблемы закрыть канал после записи в него значений  

    close(ch)
  }()

  for n := range ch {
    println(n)
  }
}
```
